Module cap.asr

Automatic Speech Recognition module.

This Module is for working with ASR (Automatic Speech Recognition) files formatted as JSON. It provides the ASR class and the Word and Punc dataclasses.

Example

>>> import asr
>>> asr.ASR('/path/to/file.json').groups()
[Word(text='An', start=0, end=1, weight=0),
 Word(text='example', start=1, end=2, weight=0),
 Punc(text='.', start=2, end=2, weight=0)]
